完整代码如下：

#include <iostream>
using namespace std;

const int N = 1e3 + 10, mod = 1e8 + 7;

int get_mod(int a, int n)
{
    return (a % n + n) % n;
}

int f[N][N];

int n, s, a, b;

int main()
{
    cin >> n >> s >> a >> b;
    f[0][0] = 1;
    for(int i = 1; i <= n - 1; i++)
    {
        for(int j = 0; j < n; j++)
        {
            f[i][j] = get_mod(f[i][j] + f[i - 1][get_mod(j - (n - i) * a, n)], mod);
            f[i][j] = get_mod(f[i][j] + f[i - 1][get_mod(j + (n - i) * b, n)], mod);
        }
    }
    cout << f[n - 1][get_mod(s, n)] << endl;
    return 0;
}

将长度为 NNN 的序列分段，保证每段和不超过 MMM 的情况下，求每段中「所有数的最大值」之和的最小值

原题链接：AcWing 299. 裁剪序列

直观考虑，一共 NNN 个数，之间的空位有 N−1N-1N−1 个，因此可以分段的选择一共有 2n−12^{n-1}2n−1 ，考虑动态规划进行优化

设 f[i]f[i]f[i] 所表示的集合为：所有将前 iii 个数划分方案的集合，集合的属性为价值的最小值，因此 f[n]f[n]f[n] 为最终答案

以最后一段的长度对整个集合进行划分。对序列 f[i]f[i]f[i] 而言，设最后一段的长度为 k(0≤k≤i)k(0\le k\le i)k(0≤k≤i)，此时有：

f[i]=min⁡0≤k≤i{f[i−k]+max⁡i−k+1≤j≤i{Aj}}f[i]=\min_{0\le k\le i}\{f[i-k]+ \max_{i-k+1\le j\le i}\{ A_j\}\}f[i]=0≤k≤imin​{f[i−k]+i−k+1≤j≤imax​{Aj​}}

设 j=i−kj=i-kj=i−k ，上式转换为：

f[i]=min⁡0≤j≤i{f[j]+max⁡j+1≤k≤i{Ak}}f[i]=\min_{0\le j\le i}\{f[j]+\max_{j+1\le k\le i}\{A_k\}\}f[i]=0≤j≤imin​{f[j]+j+1≤k≤imax​{Ak​}}

容易注意到以下性质：

  • f[i]f[i]f[i] 随着 iii 的增大而单调不减

证明：

假定存在两个序列 k1, k2k_1,\ k_2k1​, k2​ ，长度分别为 L1, L2L_1,\ L_2L1​, L2​ ，有 L2>L1L_2\gt L_1L2​>L1​

我们将 k2k_2k2​ 的划分方案平移到 k1k_1k1​ 上，设划分段数为 lenlenlen ，有:len2≥len1len_2\ge len_1len2​≥len1​ ，因此对于 k2k_2k2​ 而言，必然有 f[k2]≥f[k1]f[k_2]\ge f[k_1]f[k2​]≥f[k1​] ，即 f[i]≤f[i+1]f[i]\le f[i+1]f[i]≤f[i+1]

其次，局部最优值 f[j]+Akf[j]+A_kf[j]+Ak​ 合法的充要条件为：

  • ∑k=j+1iAk≤m\sum_{k=j+1}^{i}A_k\le m∑k=j+1i​Ak​≤m

第一条保证从 j+1j+1j+1 到 iii 的和全部小于 mmm

第二条保证 jjj 总会取到总和小于 mmm 的边界

第三条保证 AkA_kAk​ 为 j+1j+1j+1 到 iii 中所有数的最大值

下面我们给出第二条的证明（第一和第三可以直接从题目推出来）：

考虑反证法，设存在 j′<jj'\lt jj′<j 此时有：f[j′]+max⁡j′<k≤i{Ak}≤f[j]+max⁡j<k≤i{Ak}f[j']+\max_{j'\lt k\le i}\{A_k\}\le f[j]+\max_{j\lt k\le i}\{A_k\}f[j′]+maxj′<k≤i​{Ak​}≤f[j]+maxj<k≤i​{Ak​}

由于 f[i]f[i]f[i] 随 iii 增大而单调不减，即 f[j′]≥f[j]f[j']\ge f[j]f[j′]≥f[j]

且 max⁡j′<k≤i{Ak}≥max⁡j<k≤i{Ak}\max_{j'\lt k\le i}\{A_k\}\ge \max_{j\lt k\le i}\{A_k\}maxj′<k≤i​{Ak​}≥maxj<k≤i​{Ak​} ，因此上述假设不成立

因此若 j′<jj'\lt jj′<j ，并且 Aj′<AjA_{j'}\lt A_jAj′​<Aj​ ，那么 j′j'j′ 就是需要淘汰的策略

此时对于区间 [j,i][j,i][j,i] 而言，内部元素单调不减且 i,ji,ji,j 均具有单调性 (i,ji,ji,j会同步增大) 也就是「双指针」与「单调队列」

其次，由于单调队列中的所有值均是局部最优解，需要全部考虑在内再取最小值，因此我们需要额外维护一个允许出现重复元素的「平衡树」用于存储每个元素所对应的函数值，每次取出最小值即可

在这里我们需要注意一个边界问题，那就是只要当队列中至少存在一个元素时，才能够开始往 multiset 中插入元素

这是因为单调队列中单调不增的元素实际上表示的是边界，也就是 f[i]f[i]f[i] 在此处的取值，而对与第二项的最大值而言，需要至少存在两个元素才合法，因此 multiset 中的元素个数总会比单调队列中少一个

完整代码如下：

#include <iostream>
#include <cstring>
#include <algorithm>
#include <set>

using namespace std;

typedef long long LL;

const int N = 1e5 + 10;

int w[N], q[N];
LL f[N];
multiset<LL>S;
LL n, m;

//multiset会直接将所有相同的元素全部删除，因此需要迭代器
void remove(int x)
{
    auto it = S.find(x);
    S.erase(it);
}

int main()
{
    cin >> n >> m;
    for(int i = 1; i <= n; i ++) 
    {
        cin >> w[i];
        if(w[i] > m)
        {
            cout << "-1" << endl;
            return 0;
        }
    }
    
    int hh = 0, tt = -1;
    LL sum = 0;
    for(int i = 1, j = 1; i <= n; i ++)
    {
        sum += w[i];
        while(sum > m)
        {
            sum -= w[j++];
            if(hh <= tt && q[hh] < j)
            {
                if(hh < tt) //保证队列中至少一个元素之后再去删除set中的元素
                    remove(f[q[hh]] + w[q[hh + 1]]);//此时q[hh]为边界，区间最大值需要取后一个元素
                hh++;
            }
        }
        
        while(hh <= tt && w[q[tt]] <= w[i])
        {
            if(hh < tt)
                remove(f[q[tt - 1]] + w[q[tt]]);//此时队尾元素表示局部最大值，边界需要取前一个元素
            tt--;
        }
        
        q[++tt] = i;        
        if(hh < tt) //以当前队头前一个元素为边界，当前队头认为是整个区间的最大值
          S.insert(f[q[tt - 1]] + w[q[tt]]);

        f[i] = f[j - 1] + w[q[hh]];
        if(S.size()) //当平衡树中不空时，我们取整个的最小值
          f[i] = min(f[i], *S.begin());
    }
    cout << f[n] << endl;
    return 0;
}
